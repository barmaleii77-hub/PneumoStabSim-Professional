## ?? Установка

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/barmaleii77-hub/NewRepo2.git
   cd NewRepo2
   ```

2. Создать виртуальное окружение:
   ```bash
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1  # Windows
   source .venv/bin/activate     # Linux/Mac
   ```

3. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## ?? Основные возможности

### Русифицированный UI:
- **Главное окно** - меню, тулбар, статус-бар на русском
- **Геометрия** - настройка параметров автомобиля
- **Пневмосистема** - управление давлением и клапанами
- **Режимы** - выбор кинематики/динамики, дорожное воздействие

### QComboBox пресеты:
1. **Геометрия** - 4 варианта (Седан, Хэтчбек, Кроссовер, Пользовательский)
2. **Пневматика** - Единицы давления (бар, Па, кПа, МПа)
3. **Режимы** - 5 вариантов (Стандартный, Кинематика, Динамика, Тест, Пользовательский)

### Виджеты управления:
- **Knob** - Поворотные регуляторы (крутилки)
- **RangeSlider** - Слайдеры с диапазоном
- **QScrollArea** - Прокрутка при переполнении
