# check_git_sync.ps1 — быстрый аудит синхронизации

`check_git_sync.ps1` выполняет быстрый аудит репозитория перед коммитом или пушем. Скрипт устойчив к отсутствию remote и сообщает, как настроить `origin` перед сравнением веток.

## Использование
```powershell
pwsh ./check_git_sync.ps1
```

## Что проверяется
1. **Платформа** — выводит описание текущей ОС для кроссплатформенного логирования.
2. **Текущая ветка** и последний локальный коммит.
3. **Состояние рабочей директории** (`git status --short`).
4. **Наличие remote** — пропускает `fetch`, если `origin` не сконфигурирован, и подсказывает команду для добавления.
5. **Сравнение с origin** — вычисляет отставание/опережение, либо предупреждает, что удалённая ветка ещё не создана.
6. **Удалённые ветки** — выводит список при наличии remotes.
7. **Последние коммиты** — последние 5 записей для быстрой диагностики.

## Примечания
- Скрипт использует только встроенные команды Git и не требует дополнительных зависимостей.
- При отсутствии remotes команда завершает проверку без ошибок и предлагает добавить `origin` для полноценной синхронизации.
