# Release Notes — HDR Bloom/Tonemap Integration (v2.0.1)

## Overview
Обновление объединяет ветку `hdr/bloom-tonemap` в основную линию разработки и
фиксирует ключевые улучшения HDR-пайплайна для релиза v2.0.1.

## Highlights
- Добавлены отдельные ползунки `HDR Maximum` и `HDR Scale` рядом с порогом
  тонмаппинга, все значения сохраняются через сервис настроек и восстанавливают
  UX для тонкой настройки динамического диапазона.
- Панель окружения использует `FileCyclerWidget` для перебора HDR skybox-файлов,
  предоставляет кнопку очистки выбора и предупреждает о ресурсе, если он
  отсутствует в манифесте или на диске.
- Главное окно нормализует локальные file URL при выборе HDR-текстур и логирует
  каждую неудачную попытку резолвинга, что облегчает кроссплатформенную отладку
  и аудит путей.

## Documentation & Verification
- HDR-руководства обновлены до версии v2.0.1, добавлены примеры кеширования и
  ссылки на манифестный пайплайн с проверкой `hdr-verify`.
- Номер релиза в стабилизационном плане и индексах ассистентов приведён к
  v2.0.1, чтобы тесты согласованности версий оставались зелёными.
