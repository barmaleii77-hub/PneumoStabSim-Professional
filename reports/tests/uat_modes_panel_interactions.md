# UAT – панель режимов и дорожных взаимодействий

## Чек-лист
- [x] Кнопки управления (старт/пауза/сброс) меняют состояние и шлют события `simulation_control`.
- [x] Переключение радиокнопок «Кинематика/Динамика» и «Изотермический/Адиабатический» обновляет пресет на «Пользовательский» и эмитит `mode_changed`.
- [x] Отключение пневматики в PhysicsTab обновляет payload `physics_options` и синхронизируется с SettingsManager.
- [x] Поля RoadExcitationTab отображают ошибки валидации (амплитуда, частота, фазы, сглаживание) после ручного ввода за пределами диапазона.
- [x] Сглаживание дорожного профиля можно включать/выключать без блокировки остальных контролов; ограничения по длительности/углу/ходу применяются автоматически.

## Ручные сценарии
1. **Старт/пауза симуляции**  
   Шаги: открыть ModesPanel → нажать «Старт», затем «Пауза».  
   Результат: статус меняется на «▶ Запущено» и «⏸ Пауза», кнопки `Старт`/`Пауза` блокируются/разблокируются, сигнал `simulation_control` приходит дважды.  
   Статус: ✅ Успех.

2. **Переключение режимов и пневматики**  
   Шаги: выбрать «Динамика» и «Адиабатический», затем снять чекбокс «Включить пневматику».  
   Результат: пресет переключается на «Пользовательский», события `mode_changed` фиксируют оба режима, payload `physics_options` содержит `include_pneumatics: false`.  
   Статус: ✅ Успех.

3. **Валидация дорожных параметров**  
   Шаги: вручную задать амплитуду 0.35 м, частоту 14 Гц, фазу –5°, длительность сглаживания 1200 мс через SettingsManager → перезагрузить вкладку дороги.  
   Результат: список ошибок включает выход за диапазон для всех параметров, UI ограничивает значения ползунков до предельных (0.2 м, 600 мс).  
   Статус: ✅ Успех.

4. **Сглаживание и синхронизация анимации**  
   Шаги: отключить чекбокс «Сглаживание анимации», затем выбрать easing `OutQuad` и снова включить.  
   Результат: контролы сглаживания блокируются/разблокируются в соответствии с чекбоксом, событие `animation_changed` обновляется с новым easing, значения длительности/порога сохраняются.  
   Статус: ✅ Успех.
