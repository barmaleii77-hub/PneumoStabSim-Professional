# 🚀 НАЧНИТЕ РАБОТУ ПРЯМО СЕЙЧАС!

## Способ 1: Нажмите F5 в VS Code ⚡

1. Откройте файл `app.py`
2. Нажмите клавишу **F5**
3. Готово! Приложение запущено!

## Способ 2: Используйте PowerShell скрипт

```powershell
.\run.ps1
```

## Способ 3: Batch файл (двойной клик)

```
run.bat
```

---

# 📋 Доступные F5 конфигурации

После нажатия F5 выберите одну из конфигураций:

- **F5: PneumoStabSim (Главный)** ← ОСНОВНОЙ ЗАПУСК
- F5: Verbose (подробные логи) - с Qt debug выводом
- F5: Test Mode (автозакрытие 5с) - тестовый режим
- F5: Verbose + Test (5с) - комбинация
- F5: Current File (текущий .py) - запуск открытого файла
- F5: Run Tests (pytest) - запуск всех тестов

---

# 🔧 Полезные команды в терминале VS Code

При открытии нового терминала доступны:

```powershell
run              # Запустить приложение
run -Verbose     # С подробными логами
run -Test        # Тестовый режим (5 сек)
test             # Запустить тесты
fmt              # Форматировать код
typecheck        # Проверить типы
clean            # Очистить кэш
info             # Показать справку
```
